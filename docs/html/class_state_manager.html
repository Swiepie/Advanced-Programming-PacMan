Detailed Description

Beheert de stack van game states en state transities.

Implementeert een stack-gebaseerd state management systeem voor het spel. Ondersteunt push/pop operaties voor state transities en bevat input buffering om ongewenste dubbele inputs te voorkomen. De actieve state staat bovenaan de stack.
