Detailed Description

Abstracte basisklasse voor het renderen van entiteiten.

Implementeert het Observer pattern om automatisch te updaten wanneer de onderliggende entity verandert. Biedt gemeenschappelijke functionaliteit voor sprite management, texture updates en frame timing voor alle entity views.
