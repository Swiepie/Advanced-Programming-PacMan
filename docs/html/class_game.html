Detailed Description

Hoofdklasse voor het opstarten en draaien van het spel.

Deze klasse initialiseert het SFML venster, de StateManager en de game-loop. Het beheert de tijdsberekeningen via Stopwatch en verwerkt window events. De Game-klasse scheidt volledig de representatie (SFML) van de game-logica.
