Detailed Description

Singleton klasse voor het genereren van willekeurige getallen.

Biedt functionaliteit voor het genereren van willekeurige integers, floats en richtingen. Gebruikt het Singleton patroon om consistente randomisatie door het hele spel te garanderen.
