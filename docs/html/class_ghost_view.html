Detailed Description

Basisklasse voor het renderen van Ghost entiteiten.

Beheert de visuele representatie van ghostsin het spel. Implementeert animaties voor verschillende richtingen en fear mode. Erft van EntityView en biedt basis functionaliteit die door specifieke ghost views uitgebreid wordt.
