Detailed Description

Abstracte basis klasse voor het State pattern in het spel.

Definieert de interface voor alle game states zoals MenuState, LevelState en PausedState. Implementeert het State pattern voor modulair state management. Elke concrete state moet de pure virtual methodes implementeren.
