Detailed Description

View klasse voor het renderen van de Pacman speler entity.

Beheert de visuele representatie van Pacman in het spel. Implementeert mond animaties die synchroniseren met beweging en richting. Erft van EntityView en implementeert het Observer pattern voor automatische updates.
