Detailed Description

State klasse voor het pauze scherm van het spel.

Toont een pauze scherm wanneer het spel gepauzeerd is. Erft van State en implementeert het State pattern voor spel flow management. Biedt de mogelijkheid om het spel te hervatten via spatiebalk.
