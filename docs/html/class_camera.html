Detailed Description

Singleton klasse voor camera en scherm transformatie beheer.

Beheert de conversie tussen game wereld coördinaten en scherm coördinaten. Gebruikt het Singleton pattern om één globale camera instantie te garanderen. Biedt hulpmiddelen voor scherm berekeningen en aspect ratio beheer.
